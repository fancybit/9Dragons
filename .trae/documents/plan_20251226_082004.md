## 3. 批量更新策略

### 3.1 PowerShell批量处理
```powershell
# 进入章节目录
Set-Location "f:/Works/九王爷/九龙修真记/md/章节/正传"

# 统一替换朝代和主要人物
foreach ($file in Get-ChildItem "*.md") {
    $content = Get-Content -Path $file.FullName -Raw -Encoding UTF8
    $content = $content -replace "大胤王朝", "大明万历"
    $content = $content -replace "萧珩", "朱常澜"
    $content = $content -replace "萧瑾", "朱常洛"
    $content = $content -replace "萧煜", "朱常洵"
    $content = $content -replace "皇后(?!娘娘)", "郑贵妃"
    $content = $content -replace "皇后娘娘", "郑贵妃"
    $content = $content -replace "坤宁宫", "翊坤宫" -replace "翊坤宫", "翊坤宫"
    $content = $content -replace "章和十七年", "万历二十五年"  # 根据情节合理设置年份
    Set-Content -Path $file.FullName -Value $content -Encoding UTF8
}
```
