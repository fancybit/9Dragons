八、描写要求

1. 场景描写要求

1.1 时代背景融合

  • 明朝万历年间的社会风貌与修真元素自然融合，避免生硬拼接
  • 皇宫、王府等建筑风格符合明朝规制，同时加入修真界特有的灵韵
  • 市井场景需体现明朝市井生活的烟火气，同时暗藏修真界的隐秘线索
  • 修真场所（如青云观、菘蓝社、幽冥界）需突出其神秘性和独特的空间感

1.2 地域特色鲜明

  • 济南、兖州等济水流域场景需体现齐鲁文化特色和水韵
  • 昆仑、雪灵谷等修真圣地需突出其险峻、清幽或神秘的自然环境
  • 不同地域的建筑风格、植被、气候需符合现实地理特征

1.3 氛围营造

  • 皇宫场景：威严庄重中暗藏权谋斗争的紧张感
  • 王府场景：宁静雅致中透出修真者的超脱气质
  • 修真战斗场景：气势磅礴，天地变色，体现道空力量的碰撞
  • 市井场景：热闹喧嚣，充满生活气息，与修真界形成鲜明对比

1.4 细节描写

  • 注重场景中的细节元素，如宫灯的样式、王府的匾额、修真场所的符篆等
  • 场景中的气味、声音、光线需细致描写，增强沉浸感
  • 场景中的动态元素（如飘动的旗帜、流动的灵韵、行人的动作）需生动刻画

2. 物品描写需求

2.1 武器描写

  • 注重武器的材质、工艺、纹饰和特殊能力
  • 武器与使用者的契合度需有所体现
  • 武器在不同场景下的表现（如战斗中、静置时）需有区别
  • 传统武器（如唐刀、苗刀）需符合历史形制，同时加入修真元素

2.2 道具描写

  • 修真道具（如本命灵珠、九龙玉佩）需突出其神秘力量和历史渊源
  • 日常物品需体现明朝生活特色，同时可能暗藏修真线索
  • 物品的来历、传承和意义需适当交代，增强其故事性

2.3 灵材描写

  • 灵材的外观、气味、触感需细致刻画
  • 灵材的生长环境和特性需符合修真逻辑
  • 灵材在使用过程中的变化和效果需生动呈现

2.4 服饰描写

  • 明朝服饰需符合历史规制，不同身份的服饰需有所区别
  • 修真者的服饰需体现其修为境界和门派特色
  • 服饰的材质、纹饰、色彩需与人物性格和身份相符

3. 人物描写需求

3.1 外貌描写

  • 注重人物的面部特征、气质和细节（如眼神、表情、手势）
  • 人物外貌需与性格、身份、修为境界相符
  • 妖族人物的外貌需体现其种族特征，同时保持美感
  • 修真者的外貌需体现其修真者的特质，如气场、灵韵等

3.2 性格刻画

  • 通过言行举止、心理活动、与他人的互动等多维度刻画人物性格
  • 人物性格需有层次感和成长变化
  • 反派人物需避免扁平化，体现其复杂性和动机
  • 主角朱常澜的性格需突出其温润如玉却内心坚韧的特质

3.3 身份与地位体现

  • 皇室成员（如万历皇帝、太子朱常洛）需体现其威严和权谋
  • 修真者需体现其修为境界和门派地位
  • 济王朱常澜需体现其王爷身份与修真者身份的双重特质

3.4 情感表达

  • 人物情感需真实自然，符合其性格和处境
  • 朱常澜与赵玉珏的感情需体现其深厚和默契
  • 兄弟情（如朱常澜与朱常洛）需真诚感人
  • 师徒情、友情等其他情感需丰富多样

3.5 时间与年龄描写

  • 人物的外貌、行为举止需符合其实际年龄，如少年时期的青涩、青年时期的成熟、老年时期的沉稳等
  • 人物的成长经历需与时间线保持一致，如朱常澜从童年到成年的变化、修为境界的提升等
  • 重要事件的发生时间需符合历史背景和小说设定，如万历皇帝封朱常澜为济王的时间、朱常洛即位的时间等
  • 人物的出生年月需与历史记载或小说设定相符，如万历皇帝出生于1563年、朱常洛出生于1582年等
  • 事件之间的时间间隔需合理，符合逻辑，如从东宫遇刺到朱常澜被封为济王的时间跨度

4. 战斗描写需求

4.1 战斗风格

  • 朱常澜的战斗需体现其太极拳、八极拳和戚家刀的特色，以及修真者的道术
  • 不同门派的战斗风格需有所区别（如青云观的道家法术、南少林的佛家武学）
  • 妖族的战斗需体现其种族特性和本能
  • 战斗需符合修真逻辑，避免过于夸张或违背设定

4.2 境界与战斗能力

  • 解经境：习得武学基础，战斗以基础招式为主，法术道术较为简单。能够初步运用灵气，战斗方式较为单一，主要依靠自身力量和基础招式，如犬族/星瑶、狐妖/林栖梧等解经境后期修士，战斗时以基础技能和本能为主。
  • 凝则境：习得各种高级技巧，招式更加精妙，可使用中级法术道术。能够熟练运用灵气，战斗方式多样化，开始注重战术和技巧，如赵玉珏/赵小月、鬼王/鬼椛、毒医少女/苏小棠、耿直剑修/林长风、腹黑术师/楚轻寒、戏腔艺人/猫三三等凝则境修士，战斗时展现出精湛的技艺和多样化的战斗方式。
  • 化道境：可以融合不同武学，实现简单小队配合，能使用高级法术道术。能够将灵气与自身武学融合，形成独特的战斗风格，开始懂得运用天地之力，如穆清瑶/穆桂英转世、玄清道长/青云观观主、玄宸/墟影教教主、社长/谪危等化道境修士，战斗时能够融合多种武学，实现小队配合，展现出强大的个人实力和团队协作能力。
  • 合天境：可以组织战术战阵，组织大规模战斗，甚至使用控制天气、地气、人心的大规模法术道术。能够与天地融为一体，运用天地之力进行战斗，战斗方式已经超越了个人层面，能够影响整个战场，如玄渊/玄渊妖君等合天境修士，战斗时能够组织大规模战阵，操控天地之力，展现出毁天灭地的强大实力。

4.3 动作描写

  • 战斗动作需流畅连贯，富有节奏感
  • 注重武器的使用技巧和招式名称（如辛酉刀法、五行唐刀等）
  • 修真法术的施展过程需详细描写，包括手势、咒语、效果等
  • 战斗中的动态变化需及时呈现，如攻防转换、招式变化等

4.4 战斗氛围

  • 战斗场景需营造紧张刺激的氛围
  • 不同规模的战斗（如个人对决、团体战、三界大战）需有不同的氛围营造
  • 战斗中的环境变化（如天地变色、灵韵波动）需与战斗进程相呼应
  • 战斗中的心理描写需体现人物的策略、决心和情感变化

4.5 效果描写

  • 战斗中的物理效果（如刀光、拳风、爆炸）需生动刻画
  • 修真法术的效果（如灵韵流动、空间扭曲、封印阵激活）需富有想象力
  • 战斗对环境的破坏需符合逻辑，增强真实感
  • 战斗对人物的影响（如伤势、修为变化、心理冲击）需有所体现

4.6 战术与策略

  • 战斗中的战术运用需符合人物性格和修为境界
  • 不同人物的战斗策略需有所区别（如朱常澜的以柔克刚、楚轻寒的腹黑狡诈）
  • 团队战斗需体现成员之间的配合和互补，根据成员境界合理安排战术
  • 战斗中的反转和意外需合理安排，增强戏剧性
